# AI-Sentiment-Analyser

## Overview
AI Sentiment Analyzer is a web-based application that classifies customer reviews into positive, negative, or neutral sentiments using a state-of-the-art transformer model. Built with Streamlit and HuggingFaceâ€™s DistilBERT model, this tool enables real-time, accurate sentiment detection and provides an intuitive user interface for seamless interaction.

## Features
- Real-time sentiment analysis of user-input text
- Accurate classification using DistilBERT fine-tuned on SST-2 dataset
- User-friendly and visually appealing web interface built with Streamlit
- Displays sentiment label with confidence score and expressive emojis
- Fast processing with near-instant results
